# רז עייני - סקריפטים

### 1. יצירת סיסמה רנדומלית מאובטחת
* **מה זה עושה:** מייצר סיסמה חזקה באורך 10 תווים.
* **איך זה עובד:** שולף תווים אקראיים מ-`/dev/urandom`, מבטיח לפחות אות גדולה, אות קטנה, מספר וסימן מיוחד, ומערבב אותם בעזרת `shuf`.

### 2. סריקת פורטים פתוחים (Port Scanner) בכתובת IP
* **מה זה עושה:** בודק אילו פורטים פתוחים להאזנה בכתובת IP מוגדרת.
* **איך זה עובד:** משתמש במנגנון ה-TCP המובנה של Bash (`/dev/tcp/ip/port`) כדי לבדוק חיבור במהירות ללא צורך בכלים חיצוניים.

### 3. הוספת תחילית לקבצי TXT בתיקייה
* **מה זה עושה:** משנה שם המוני (Bulk Rename) לכל קבצי ה-`.txt` בתיקייה נתונה.
* **איך זה עובד:** רץ בלולאה על התיקייה ומוסיף את הקידומת שבחרת לתחילת שם הקובץ באמצעות פקודת `mv`.

### 4. הצגת סטטוס Git לכל תת-תיקייה
* **מה זה עושה:** מציג במרוכז את מצב ה-Git של כל התיקיות שנמצאות בתוך תיקיית האב.
* **איך זה עובד:** מזהה תיקיות עם ספרייית `.git` נסתרת ומריץ עליהן `git -C <path> status -s` כדי לקבל פלט קומפקטי מבלי להחליף תיקייה.

### 5. בדיקת תקינות גישה (HTTP Status) לרשימת אתרים
* **מה זה עושה:** בודק אם אתרי אינטרנט זמינים ומציג את קוד הסטטוס שלהם.
* **איך זה עובד:** שולח בקשת כותרים מהירה (`curl --head`) ומציג אינדיקציה ויזואלית (`✅` או `❌`) לפי קוד ה-HTTP שחוזר.
